# 教程介绍

欢迎来到盐城工学院 Linux/开源同好会（盐工LUG）！让我们在 5 分钟内快速了解 **Docusaurus**。

## 开始使用

通过 **创建新站点** 开始使用。

或者 **立即尝试 Docusaurus** 通过 **[docusaurus.new](https://docusaurus.new)**。

### 你需要的工具

- [Node.js](https://nodejs.org/en/download/) 版本 20.0 或以上：
  - 安装 Node.js 时，建议勾选所有与依赖项相关的复选框。

## 生成新站点

使用 **classic 模板** 生成新的 Docusaurus 站点。

运行以下命令后，classic 模板将自动添加到您的项目中：

```bash
npm init docusaurus@latest my-website classic
```

你可以在命令行提示符、PowerShell、终端或任何代码编辑器的集成终端中输入此命令。

该命令还会安装运行 Docusaurus 所需的所有必要依赖项。

## 启动你的站点

运行开发服务器：

```bash
cd my-website
npm run start
```

`cd` 命令用于更改你的工作目录。为了使用新创建的 Docusaurus 站点，你需要在终端中导航到该目录。

`npm run start` 命令在本地构建你的网站，并通过开发服务器提供服务，准备在 http://localhost:3000/ 查看。

打开 `docs/intro.md`（本页面）并编辑一些行：站点 **会自动重新加载** 并显示你的更改。
