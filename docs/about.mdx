# 关于我们

欢迎加入盐城工学院 Linux/开源同好会（YCIT-LUG）！

## 这是什么？

**盐城工学院 Linux/开源同好会** 是一个致力于推广和研究 Linux 及开源文化的学生组织（非官方）。我们通过技术分享、讨论和实践，帮助同学们了解和掌握开源技术。

## 我们的目标

- **推广 Linux** - 传播 Linux 操作系统的知识和应用
- **倡导开源** - 推广开源精神和开源软件的使用
- **技术交流** - 提供一个平台让大家分享技术经验
- **社区建设** - 建立一个友好、互助的技术社区

## 我们做什么

- **技术分享会** - 定期举办讲座和研讨会
- **项目实践** - 参与和组织开源项目
- **讨论交流** - 在线和离线的技术讨论
- **资源共享** - 提供学习资源和技术文档

## 加入我们

如果你对 Linux 或开源感兴趣，欢迎加入我们！无论你是初学者还是经验丰富的开发者，我们都欢迎你的加入。
