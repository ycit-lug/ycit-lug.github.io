# 使用说明

本页面为您介绍如何使用本官网及其各项功能。

## 网站导航

### 📚 教程

点击顶部导航栏中的 **教程** 链接，可以查看详细的技术教程和指南。这里包含：
- Docusaurus 快速入门
- Linux 基础知识
- 开源项目指南

### 📝 博客

点击顶部导航栏中的 **博客** 链接，可以阅读我们的最新文章和新闻。社团成员会在这里分享：
- 技术心得
- 活动总结
- 项目进展

### 👥 关于我们

了解更多关于盐工LUG的信息，包括我们的目标、活动内容和联系方式。

## 如何浏览

1. **使用搜索** - 点击右上角的搜索图标来快速查找内容
2. **浏览侧边栏** - 左侧侧边栏显示当前部分的所有文章
3. **返回上级** - 点击面包屑导航返回上一级
4. **深色/浅色模式** - 点击右上角的主题切换按钮改变外观

## 常见问题

### 如何联系我们？

请访问 [关于我们](/docs/about) 页面查看联系方式。

### 如何参与贡献？

我们欢迎所有形式的贡献！您可以：
- 在 GitHub 上提交 Issue 或 Pull Request
- 参加我们的讨论和分享会
- 撰写文章或技术教程

### 如何成为社团成员？

请通过联系方式与我们取得联系，我们会为您介绍社团的更多信息。

## 技术支持

如果您在浏览网站时遇到问题，请：
1. 确保您的浏览器是最新版本
2. 尝试清除浏览器缓存
3. 如问题仍然存在，请通过 GitHub 提交 Issue
